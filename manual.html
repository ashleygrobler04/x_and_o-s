Welcome!

Welcome to the verry basic game of tic tac toe.
The aimb of this game is to be the first to get 3 x's (AKA. Crosses) or O's (AKA. circles) in a horizontal, vertical or diagonal line. If you do so, you win the game.

Controls

  • In a menu, the up and down arrows move up and down.
  • In game, the up, down, left and right arrows move you around on the grid.
  • The enter key places an object on the grid if it's your turn.
  • The C key tells you who's turn it is.
  • Finaly Q returns you to the main menu.

Notes

Please note that this is a project in beta. So bugs can be expected.
For now, you can only use x's or crosses in order to play the game, the oponent will use O's or circles. This will change.
You can either play against the computer, or 2 players on the same computer.
Last note, the game only works with NVDA. Other screen readers will be implemented at a later stage
